Static positions as per html flow. These eleements are considered to be not positioned.
Relative position is the same as static, unless you give it extra values.
Giving values to top,right, bottom, & left will move the element. These values are relative to its default position.
Behaves like a relatice except that it is relative to it's parent element. Top, right, bottom, & left give it a new position.
Fixed position will move to a new position and stay there even is content around it is being scrolled through.
